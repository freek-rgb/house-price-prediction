Setup instructions
1. Place index.html and app.py in the same folder as forest_reg.pkl
2. Install dependencies: pip install fastapi uvicorn scikit-learn pandas joblib
3. Start the server: uvicorn app:app --reload --port 8000
4. Open index.html in your browser — the green dot above confirms it's connected.
